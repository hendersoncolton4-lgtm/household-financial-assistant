# Household Financial Assistant

A private, AI-powered financial overview, planner, and assistant. Built for **one household — me and my wife.** Not a product, not for sale, not for use by anyone else.

## What this is

A local-first web app that connects to our own bank accounts via [Plaid](https://plaid.com) and uses an AI assistant ([Anthropic Claude](https://www.anthropic.com)) to help us:

- See our full financial picture in one place (accounts, transactions, net worth)
- Set and track goals (emergency fund, debt payoff, retirement, savings rate, custom)
- Project long-horizon retirement scenarios with compound growth
- Generate on-demand monthly reviews of spending and progress
- Ask plain-English questions of an assistant that has live context on all of the above

Everything runs locally on a machine we control. All financial data is stored in a local SQLite database. Plaid access tokens are encrypted at rest with AES-256-GCM.

## What this isn't

- **Not a product, SaaS, or service for anyone else.** It will never accept third-party users.
- **Not a regulated financial advisor.** Any material decision (real-money trade, tax position, Roth conversion, estate document, insurance change) is something we discuss with a CPA, attorney, or fee-only fiduciary. The AI's role is to surface questions and frame trade-offs, not to give professional advice.
- **Not for distribution.** The code is public so it can be referenced (e.g., for Plaid Development access), but the running instance, its data, and its API credentials never leave the household.

## Tech stack

- **Framework:** [Next.js 16](https://nextjs.org/) (App Router) + TypeScript
- **UI:** Tailwind CSS v4, dark mode by default
- **Database:** SQLite via [better-sqlite3](https://github.com/WiseLibs/better-sqlite3) + [Drizzle ORM](https://orm.drizzle.team/)
- **AI:** [Anthropic Claude](https://www.anthropic.com) via `@anthropic-ai/sdk`, model `claude-sonnet-4-6`
- **Account aggregation:** [Plaid](https://plaid.com) (`plaid` Node SDK + `react-plaid-link`)
- **Encryption:** Node's built-in `crypto` (AES-256-GCM)

## Setup (for the household running it)

1. Install Node.js 18+ and npm
2. Clone this repo and `cd` into it
3. `npm install`
4. Copy `.env.local.example` to `.env.local` and fill in:
   - `ANTHROPIC_API_KEY` — from <https://console.anthropic.com/settings/keys>
   - `PLAID_CLIENT_ID`, `PLAID_SECRET`, `PLAID_ENV` — from your Plaid dashboard
   - `APP_ENCRYPTION_KEY` — a 32-byte hex string (run `openssl rand -hex 32`)
5. `npm run dev`
6. Open <http://localhost:3000>

## Privacy & security posture

- **Local-first.** The database, the encryption key, and the Plaid access tokens never leave the machine running the app.
- **API egress is minimal and intentional.** When the AI chat or monthly review runs, account names, balances, recent transactions, and goals are sent to Anthropic for inference. Per Anthropic's policy, API data is not used to train models.
- **Encrypted at rest.** Plaid access tokens are encrypted with AES-256-GCM before being written to disk. The key lives in `.env.local`, which is gitignored.
- **No secrets in source.** `.env.local`, `data/` (the SQLite database), and `node_modules/` are all gitignored. The public repo contains only application code.

## Disclaimer

Nothing in this application or its outputs constitutes professional financial, tax, legal, or investment advice. It is a tool for personal use by a single household. Material financial decisions should involve a qualified professional — CPA, attorney, or fee-only fiduciary.

This project also assumes that the only people interacting with it are the two members of the household. It does not implement multi-tenancy, third-party authentication, or production-grade compliance controls, because it is not intended for use by anyone else.
